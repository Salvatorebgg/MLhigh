# MLhigh v15：经典图谱扩展 + 严格对应的右侧参数调节

## 这一次的核心目标
不是单独修补 Lasso，而是把“图形—参数—方法”三者之间的逻辑继续往系统化方向推进：

1. **尽可能为主要模型补充经典图**；
2. **右侧参数区只出现左侧当前图真正需要的控件**；
3. **颜色项、线条项、点大小、柱宽等与左侧对象严格一一对应**；
4. **避免右边出现左边没有的对象，也避免左边有对象而右边调不到。**

## 右侧参数区逻辑加固

现在右侧参数区按照“当前激活的图”动态判断，只显示真正适合该图的控件：

- 有柱状图：显示“柱状图宽度”“柱体间距”“变量/柱子颜色”；
- 有散点/点图：显示“点/标记大小”；
- 有折线：显示“线条宽度”；
- 有普通系列图：显示“系列颜色”；
- 有热图/等高线图：不再错误地显示逐项系列颜色，而是明确提示此类图当前不做逐项调色；
- 所有图都可调：字体、背景、整体透明度、标题。

## 左右严格对应的改动

### 已处理
- 柱状图颜色 → 对应左侧每个柱/变量；
- 折线图颜色 → 对应左侧每条线；
- 散点图颜色与点大小 → 对应左侧散点系列；
- 柱宽、柱距 → 只在柱状图出现；
- 线宽 → 只在有折线时出现；
- 点大小 → 只在有 marker 时出现；
- 坐标轴字体、标题字体、标签字体 → 直接作用于当前图；
- 网格/空白/透明背景 → 直接作用于当前图；
- 坐标轴箭头继续保留。

### 后端

- 颜色映射改为按图索引和对象索引分别存储；
- 参数修改后实时重绘当前激活图。
