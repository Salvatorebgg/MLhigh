# MLhigh v14：Lasso/岭回归高级图形与右侧参数调节全面加固

## 本轮重点

### 1. Lasso 回归高级图形补齐

Lasso 回归现在不再只显示一张简单柱状图，而是返回 5 张经典图：

1. Lasso 非零特征系数图
2. Lasso 系数路径图
3. Lasso 交叉验证误差曲线
4. Lasso 实际值与预测值图
5. Lasso 残差诊断图

这些图更符合 Lasso 建模的高级分析逻辑：
- 看变量是否被压缩为 0；
- 看不同 alpha 下系数路径；
- 看交叉验证误差如何选择最佳 alpha；
- 看预测拟合效果；
- 看残差是否存在系统性模式。

### 2. 岭回归高级图形补齐

岭回归同样返回 5 张经典图：

1. 岭回归特征系数图
2. 岭回归系数路径图
3. 岭回归交叉验证误差曲线
4. 岭回归实际值与预测值图
5. 岭回归残差诊断图

岭回归用于 L2 正则化和多重共线性缓解，重点看系数收缩路径和预测稳定性，而不是变量是否被剔除。

### 3. 多图顶栏切换

如果一个方法返回多张图，结果可视化区顶部会自动生成图形切换标签。
用户可以在同一结果可视化页中切换不同图形，而不是所有图堆在一起。

### 4. 右侧图形参数调节真正生效

右侧参数区继续对齐 jichutongji_zhong 风格，并补齐实际功能：

- 图表主题
- 图表标题
- 坐标轴刻度字体
- 坐标轴标题字体
- 标签字体
- 图形背景：网格线 / 空白 / 透明
- 柱状图宽度
- 柱体间距
- 点 / 标记大小
- 线条宽度
- 整体透明度
- 变量 / 柱子颜色逐项调节
- 系列颜色逐项调节
- 重置颜色

### 5. 变量颜色和图形对象对应

对于柱状图，如果横轴是变量名：
- 右侧颜色项显示对应变量名；
- 修改变量颜色会修改对应柱子的颜色；
- 不再出现右侧颜色项和左侧图形对象对应不上的情况。

对于折线图、散点图：
- 右侧颜色项对应具体系列；
- 点大小、线宽、透明度会作用到对应图形对象。

### 6. 坐标轴箭头

图形布局增加带箭头坐标轴效果：
- x 轴末端箭头；
- y 轴末端箭头；
- 坐标轴线条加粗；
- 保持出版风格图形观感。
