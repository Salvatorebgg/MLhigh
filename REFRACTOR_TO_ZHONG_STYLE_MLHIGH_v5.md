# MLhigh v5 数据预览界面修正

## 本轮修改

1. 数据上传界面继续保持纯数据预览逻辑
   - 不显示分析方法；
   - 不显示方法卡片；
   - 不显示方法相关统计；
   - 只负责加载/上传数据和展示数据前部预览。

2. 数据预览区域优化
   - 预览表展示全部列；
   - 默认展示前 80 行；
   - 表头固定；
   - 第一列固定；
   - 表格区域内部支持横向与纵向滚动。

3. 滚动逻辑修正
   - 整个系统界面不出现滚轮；
   - 只有数据预览表格内部出现横向/纵向滚轮；
   - 方便查看宽表和多行数据。

4. 检查
   - 前端 JS 语法检查通过；
   - 后端核心文件编译通过；
   - smoke tests 通过。
