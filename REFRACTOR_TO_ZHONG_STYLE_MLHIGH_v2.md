# MLhigh → jichutongji_zhong 风格与流程二次重构 v2

## 本轮按反馈修复

1. **流程逻辑向 jichutongji_zhong 靠拢**
   - 原流程：先选分析方法，再加载数据，再配置参数；
   - 新流程：先上传/加载数据 → 选择变量角色 → 系统判断可用方法 → 选择分析方法 → 参数配置 → 执行分析。
   - 新增“选择变量”步骤，采用四栏式变量转移：
     - 全部变量
     - 研究变量
     - 协变量 / 混杂因素
     - 结局变量

2. **方法可用性判断**
   - 系统根据变量角色与变量类型判断方法能否运行；
   - 不适用的方法自动变灰；
   - 点击不可用方法会提示原因；
   - 方法判断覆盖高级统计、机器学习和综合工具。

3. **删除分析方法卡片左侧蓝色字母标签**
   - 方法卡片只保留方法名称和可运行/不可运行说明。

4. **结果图形区域**
   - 图形等比例缩小；
   - 图形居中显示；
   - 图形参数区放到图形右侧；
   - 参数区包括图表标题、图表主题和外观参数。

5. **结果讨论样式**
   - 字体增大；
   - 删除左侧蓝色/绿色边框；
   - 改为更干净的白底报告样式。
